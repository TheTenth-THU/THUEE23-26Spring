## 课程安排

### 理论授课安排

+ 第2周：**实验绪论**，**理论讲课1**
	+ 实验课程简介
	+ 分布参数、传输线波动方程
	+ 阻抗与驻波、传输线的工作状态（行波、纯驻波、驻波）
+ 第3周：**理论讲课2**
	+ Smith圆图和阻抗匹配
	+ 同轴线、矩型波导、微带线的场型分布
+ 第8周：**理论讲课3**
	+ 散射矩阵与各参量定义
	+ 微波电路及测量

### 实验安排

+ 第4 ~ 5周：**第一次实验**
	+ 演示：电磁波在矩形波导、平行双线、同轴线中的传播特性
	+ 电磁波综合测试仪：自由空间电磁波波长的测量、矩形波导截止特性的研究
	+ 测量线系统：波导波长、驻波比和负载阻抗的测量
+ 第6 ~ 8周：**第二次实验**
	+ 演示：电磁波在微带线中的传输特性、电磁波的辐射和极化特性
	+ 电磁波综合测试仪：左/右旋圆极化波的产生与特性研究
	+ 矢量网络分析仪：不同封装电路元件的阻抗－频率特性研究；微波负载的阻抗测量和调匹配
+ 第9 ~ 11周：**第三次实验**
	+ 演示：电磁波在微波元件中的传播特性
	+ 电磁波综合测试仪：电磁波的反射、折射和极化特性研究，以及频率选择特性的研究
	+ 矢量网络分析仪：微波电路的特性研究
+ 第12 ~ 13周：**ADS仿真实验**
	+ 熟悉ADS软件的基本操作
	+ 掌握微带线工作原理
	+ 掌握微带线基本元件和匹配电路的设计

## 考核要求

实验成绩组成：
+ **平时成绩** (20')
	+ 预习报告
	+ 实验考勤
	+ 实验操作
+ **实验报告** (60')：文字表达、数据图表、实验分析和思考
+ **考核** (20')：实验原理、仪器操作
